
# Vote

  * 后端
      * cd vote-backend
      * npm i  --registry=https://registry.npmjs.com  // 安装依赖
      * node app.js //运行服务器 port=3005
      
  * 前端
      * cd vote-frontend
      * npm i  --registry=https://registry.npmjs.com  // 安装依赖
      * yarn start //启动项目

* 技术栈
  * 前端：React + react-router
  * 后端：Node.js + Express + Sequelize
